# Quickstart: 機票比價入口

## 1. 環境準備

1. 安裝 Ruby 4.0.1。
2. 執行 `bundle install`。
3. 執行 `bin/rails db:prepare`。

## 2. 啟動應用

1. 設定外部售票來源與匯率來源所需的 environment variables。
2. 啟動 Rails server：`bin/rails server`。
3. 於另一個終端啟動背景工作處理程序；若沿用 Rails 預設 queue adapter，需確認 job worker 可執行。

## 3. 核心驗證流程

1. 進入首頁並看到搜尋表單。
2. 輸入起飛機場、目的地機場、是否直飛、航程型態、出發日期區間與旅遊天數。
3. 若選擇多點進出，輸入 2 到 4 段航程。
4. 送出搜尋後，確認頁面立即顯示搜尋已建立與來源狀態區塊。
5. 等待來源逐步回傳，確認結果清單顯示：
   - 原始幣別與比較用顯示幣別
   - 含稅含必要費用的最終總價
   - 航班時間與總旅行時間
   - 購買連結
   - 最優惠推薦標示
6. 切換排序條件，確認只重排已取得結果，不重新提交搜尋。
7. 模擬單一來源失敗，確認其他來源結果仍可顯示，且失敗來源有清楚狀態。

## 4. 測試

- 執行全部測試：`bin/rails test`
- 執行 system tests：`bin/rails test:system`

## 5. 驗收重點

- 搜尋提交請求不等待所有來源完成才回應。
- 結果推薦永遠以換算後的最終總價計算。
- 多點進出最多接受 4 段，超過時表單驗證失敗。
- 無障礙要求至少覆蓋欄位標籤、鍵盤操作、錯誤提示與推薦區塊可辨識性。
